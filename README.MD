# Fundamentos NodeJS

### Pré-requisitos
- Node: 18.x.x

### Run Project

```sh
npm run dev
```

## Rotas da API

### Usuários
| Endpoint | Resposta | Resumo |
|----------|----------|--------|
| ```GET /users```| ```200``` | Retorna uma lista de usuários do bd. |
| ```POST /users```| ```201``` | Cria um novo usuário no bd. |
| ```PUT /users```| ```204``` | Atualiza um usuário da bd. |
| ```DELETE /users```| ```204``` | Deleta um usuário da bd. |
