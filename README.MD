# Fundamentos NodeJS

### Pré-requisitos
- Node: 18.x.x

### Run Project

```sh
npm run dev
```

## Rotas da API

### Usuários
| Endpoint | Resposta | Resumo |
|----------|----------|--------|
| ```GET /users```| ```200``` | Retorna uma lista de usuários |
| ```POST /users```| ```201``` | Retorna body vazio e status 201.|
